# Images SEO & PWA

Ce dossier doit contenir les images suivantes pour optimiser le SEO et le support PWA :

## Images requises

### Favicon

- **favicon.ico** (32x32) - Icône du site dans l'onglet du navigateur

### PWA Icons

- **icon-192.png** (192x192) - Icône PWA petite taille
- **icon-512.png** (512x512) - Icône PWA grande taille

### Apple Touch Icon

- **apple-touch-icon.png** (180x180) - Icône pour iOS

### Open Graph / Social Media

- **og-image.jpg** (1200x630) - Image de prévisualisation pour les réseaux sociaux

## Génération des images

Vous pouvez utiliser les outils suivants pour générer ces images :

1. **Favicon** : https://realfavicongenerator.net/
2. **PWA Icons** : Créez une image 512x512 avec votre logo
3. **OG Image** : Créez une image 1200x630 avec le logo et le nom de l'app

## Couleurs recommandées

- Couleur principale : `#4f46e5` (indigo)
- Couleur de fond : `#ffffff` (blanc)
- Couleur de texte : `#1a1a1a` (presque noir)
